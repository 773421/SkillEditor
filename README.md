# 技能编辑器
1.选择敌人类型 / 玩家类型                                                                                                                        
2.选择敌人 / 玩家 Prefab                                                                          
3.在输入框输入技能名称   点击 “创建新的技能”按钮                                                   
4.选择动画 特效  声音  填写延迟时间  倍速/暂停/播放
